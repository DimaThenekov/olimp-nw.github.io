Пятые городские состязания по робототехнике
Великий Новгород

27 ноября 2014 г. на базе ДД(Ю)Т им. Л.Голикова состоятся состязания по робототехнике в рамках городского конкурса "Планета роботов". Ребята примут участие в следующих видах состязаний: "Кегельринг", "Следование по линии", "Лабиринт", "Кегельринг-макро", "Слалом по линии", "Сумо" и "Творческая номинация".

ВНИМАНИЕ! В Положении о Шестых городских состязаниях по робототехнике (которое разослали по школам) не исправлена ссылка на электронную регистрацию и поэтому вы попали на эту страничку нажав по ссылке в полученном документе с положением. Если вы хотите зарегистрироваться на Шестые городские состязания по робототехнике, то перейдите по этой ссылке >>

Регламенты состязаний в формате Открытых состязаний Санкт-Петербурга по робототехнике:

>> Регламент состязания "Кегельринг" (для начинающих - первый год обучения)
>> Регламент состязания "Следование по линии"
>> Регламент состязания "Кегельринг-макро"
>> Регламент состязания "Слалом по линии"
>> Регламент состязания "Интеллектуальное сумо (Мини-сумо 15×15)" (протокол состязаний)

Регламенты состязаний в формате областного конкурса :

>> Регламент состязания "Лабиринт"
>> Регламент состязания "Сумо"
>> Регламент "Творческая номинация"

По итогам состязаний по регламенту областного конкурса по робототехнике и интеллектуальным системам, победители будут заявлены на участие в состязаниях Новгородской области, которые пройдут 4 декабря!

Партнеры проекта:

- Комитет по образованию Администрации Великого Новгорода;
- Центр дополнительного образования "Городская компьютерная школа" при МАОУ "Гимназия "Эврика";
- Дворец Детского и Юношеского Творчества им. Л. Голикова;
- Депутаты Думы Великого Новгорода В.А. Еремин, А.Н. Ломанов;
