Восьмые городские состязания по робототехнике
Великий Новгород

27 марта 2016 г. на базе центра "Диалог" состоятся состязания роботов в рамках городского конкурса "Планета роботов" и областного конкурса по робототехнике.
Ребята примут участие в следующих видах состязаний:
"Интеллектуальное сумо", "Кегельринг для начинающих", "Лабиринт", "Следование по линии RobotChallenge", "Творческая номинация" и "Чертежник".

Регламенты состязаний по робототехнике:

>> Регламент состязания "Интеллектуальное сумо (Мини-сумо 15×15)"
>> Регламент состязания "Следование по линии RobotChallenge (образовательные конструкторы, узкая линия)"
>> Регламент состязания "Кегельринг для начинающих"
>> Регламент состязания "Лабиринт"
>> Регламент "Творческая номинация"

>> Электронная заявка для участия в состязаниях
>> Электронная заявка для участия в творческой номинации

>> Образец бланка соглашения на обработку персональных данных
>> Положение VIII открытых состязаний по робототехнике Великого Новгорода «Планета роботов»

По итогам состязаний победители будут заявлены на участие в открытых состязаниях Санкт-Петербурга, которые пройдут 2-3 апреля!

Регламент состязаний в номинации "Чертежник" для участия в областном конкурсе:

>> Чертежник (старшая)
