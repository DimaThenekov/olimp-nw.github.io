Летняя КШ "Кэш"


В НовГУ им.Я.Мудрого проходит подготовка по программированию для победителей и призеров областной и городской олимпиады по информатике. Подготовка организована по инициативе областной школы олимпийского резерва "София" и "Городской компьютерной школы" при МАОУ "Гимназия "Эврика".

1-й день - "Переборные методы решения задач по программированию. Рекурсивная и не рекурсивная реализация перебора. Сокращение перебора. Задача о ферзе."
(08 февраля - пятница, 15.00)

результаты решения задач 08.02.13


2-й день - "Недостатки переборных методов программирования. Динамический метод решения задач. Рекурсивный и не рекурсивный вариант реализации динамического метода. Задача "Номера телефонов"".
(09 февраля - суббота, 10.00)

результаты решения задач 09.02.13


3-й день - "Граф. Основные понятия. Представление графов в памяти компьютера. Путь. Алгоритм поиска пути. Обход в "глубину"".
(10 февраля - воскресенье, 10.00)

результаты решения задач 10.02.13


4-й день - "Олимпиада по пройденным темам".
